# mathewelt_sim

Interaktive HTML-Simulation zur MatheWelt-Aufgabe „Durchschnitt als Ausgleichswert“.

## Ziel
Visualisierung der Teamwerte von Unicorns und Mathe-Asse mit Wasserbechern und Säulendiagrammen.

## Start
1. `index.html` im Browser öffnen
2. Werte eingeben oder Originalwerte laden
3. Durchschnitt zeigen oder ausgleichen

## Dateien
- `index.html`
- optional `style.css`
- optional `script.js`

## Hinweis
Dieses Tool ist als digitale Ergänzung zum MatheWelt-Arbeitsheft gedacht.
